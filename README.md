## Getting Started

### Install

create the project
```
git clone https://github.com/kimseungyeon001/react-sample-240123.git
```

access the project directory
```
cd react-sample-240123
```

install dependencies
```
npm install
```

setting .config.ts
```
cp ./src/config.local.ts ./src/config.ts
```

### Start
```
npm run dev
```

### Storybook
```
npm run storybook
```
